# MicroEscrow

Trustless escrow for secure small transactions.

## Features
- Multi-chain support (Base & Stacks)
- Secure Contract Logic
- React/Next.js Frontend

## Contracts
- Base: `contracts/base/src/MicroEscrow.sol`
- Stacks: `contracts/stacks/contracts/microescrow.clar`

## Setup
```bash
pnpm install
pnpm dev
```
